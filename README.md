# Анализ и прогноз оттока клиентов телеком-компании

## Описание проекта

Данный проект направлен на анализ и прогнозирование оттока клиентов телеком-компании с использованием методов машинного обучения. Цель — определить клиентов, которые с высокой вероятностью уйдут, и понять ключевые факторы, влияющие на это решение.

Результаты модели могут быть использованы бизнесом для разработки стратегий удержания клиентов и снижения уровня оттока.



## Бизнес-задача

Отток клиентов (churn) — это одна из ключевых проблем. Потеря клиента означает прямую потерю дохода, а привлечение нового клиента обычно обходится дороже, чем удержание существующего.

Цель проекта:
- предсказать вероятность ухода клиента
- выявить основные причины оттока
- предложить бизнес-рекомендации для снижения churn



## Данные

Используется датасет телеком-компании, содержащий информацию о клиентах:

- демографические данные (пол, наличие партнёра и иждивенцев)
- информация об услугах (интернет, телефония и дополнительные сервисы)
- тип контракта
- способ оплаты
- ежемесячные и общие расходы
- целевая переменная — факт ухода клиента (Churn)



## Этапы работы

В рамках проекта были выполнены следующие шаги:

- загрузка и первичный анализ данных
- исследовательский анализ данных (EDA)
- обработка пропусков и преобразование признаков
- feature engineering (создание новых признаков)
- обучение моделей машинного обучения
- сравнение моделей
- анализ важности признаков


## Модели

Были обучены следующие модели:

- Logistic Regression (базовая модель)
- Random Forest
- XGBoost



## Метрика качества

Для оценки качества моделей использовались:

- ROC-AUC (основная метрика)
- Precision, Recall, F1-score
- Confusion Matrix



## Результаты

Лучшая модель: Logistic Regression  
ROC-AUC: ~0.83

Результаты показывают, что модели достаточно хорошо разделяют клиентов на ушедших и не ушедших.



## Важные факторы оттока

По результатам анализа и модели, ключевыми факторами являются:

- тип контракта (особенно помесячная оплата)
- стаж клиента (новые клиенты более склонны к уходу)
- размер ежемесячных платежей
- сегмент клиентов с высоким риском (high risk)
- тип интернет-сервиса (fiber optic)



## Бизнес-выводы

Анализ показывает, что отток клиентов в основном связан не с одной причиной, а с сочетанием факторов:

- новые клиенты уходят чаще
- дорогие тарифы увеличивают вероятность оттока
- отсутствие долгосрочного контракта снижает лояльность

Рекомендации для бизнеса:

- усилить удержание клиентов в первые 6–12 месяцев
- стимулировать переход на долгосрочные контракты
- пересмотреть тарифы для высокорисковых сегментов
- улучшить качество обслуживания в проблемных сегментах
